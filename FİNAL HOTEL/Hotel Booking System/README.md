 ## Hotel Booking System 

This project is a simple hotel reservation system developed in Java using object-oriented programming principles.
It allows managing rooms, customers, reservations, and payments with file-based persistence.

## Features

Create and manage customers

View available rooms

Book and cancel reservations

Calculate total price based on room type and stay duration

Support for Standard and Suite rooms

Handle Cash and Credit Card payments

Save customer and reservation data to files

Custom exception handling

## Technologies Used

Java
Object-Oriented Programming (OOP)


## How to Run

Compile all .java files

Run the HotelGUI class

## Notes

Main class is used for testing purposes only

Customer and reservation data are stored in text files

Project is designed for educational use
